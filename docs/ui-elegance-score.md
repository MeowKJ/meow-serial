# UI Elegance Score

Meow Serial 的首页优雅度按六个维度评分。目标档位为 S。

## S 档标准

1. 资源策略优雅
   - 品牌资源来自可维护的包或 CDN 管线，不把外部资产手动塞成本地孤立文件。
   - 关键首屏资源有 `preconnect`、`preload` 或等效优化。

2. 初次使用震撼感
   - 第一屏能同时看见产品主张、真实数据预览和明确 CTA。
   - 视觉有记忆点，但不牺牲工具可读性。

3. 人类交互
   - 人类用户能立即识别“看实时图表”“打开工作台”“生成协议工作流”。
   - 按钮层级清楚，主操作和次操作不会抢焦点。

4. AI 交互
   - AI 能在首屏看到 API、Schema、data-ai 自动化入口。
   - 页面公开路径和选择器契约不依赖可见文案。

5. 可爱动画
   - 动画能增强品牌感，而不是遮挡内容。
   - 支持 `prefers-reduced-motion`，减少动态效果时仍可用。

6. 桌面响应式
   - 主屏 1280 x 800 首屏无重叠、无横向溢出、实时看板完整可见。
   - 小桌面 960 x 640 仍保持工具核心可见。
   - 宽屏保持稳定留白和双栏比例。

## 当前结论

当前目标：S。  
验收截图以 `1280 x 800` 主屏为核心，同时检查 `960 x 640` 和宽屏桌面。
